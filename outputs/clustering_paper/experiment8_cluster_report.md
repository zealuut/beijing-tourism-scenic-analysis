# 论文口径聚类报告

## 1. 主聚类方法
- 聚类对象：20 个景区
- 聚类方法：Ward 层次聚类
- 聚类特征：三个一级治理主题的 priority_index 与 dissatisfaction_index，以及 total_priority_index、mean_dissatisfaction
- 说明：景区原有类别不作为聚类输入，而作为聚类后的解释变量

### 票务高压治理型
- 景区：北京动物园、明十三陵
