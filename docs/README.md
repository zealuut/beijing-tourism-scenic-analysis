# 说明文档目录

本目录专门存放项目中的说明类文档，包括但不限于：

- 方法说明
- 口径说明
- 预处理规则说明
- 实验配置说明
- 数据替换与样本调整说明

- `全流程文档.md`：项目从原始抓取到正式版 IPA / 聚类 / POI 的整体流程说明
- `LDA降噪规则说明.md`：LDA 建模前的降噪、过滤和保留规则
- `LDA一级主题与11个二级标签对应说明.md`：一级主题与二级问题标签之间的概念对应与样本映射关系
- `11个二级标签边界说明.md`：LLM 问题标注时 11 个标签的边界与判定原则
- `SnowNLP景区领域情感打分说明.md`：SnowNLP 领域重训、阈值选择和情感三分类口径
- `论文口径IPA说明.md`：论文正式版 IPA 的指标定义、计算公式和输出说明
- `论文口径聚类说明.md`：论文正式版聚类的方法、变量、分组思路和解释口径
- `POI周边采集说明.md`：景区周边 POI 采集方法、缓冲圈口径和输出位置
- `景区边界_内部外部POI_面积说明.md`：景区边界来源、内部/外部 POI 判定规则与面积口径
- `POI指标定义说明.md`：POI 大类、计数、最近距离、多样性、内部/外部与面积指标的定义
