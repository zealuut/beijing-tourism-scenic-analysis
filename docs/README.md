# 说明文档目录

本目录用于存放项目中的方法说明、口径说明、案例说明与阅读指引。

## 核心文档

- `全流程文档.md`
  项目从评论样本、主题识别、情感评估、IPA/聚类，到回归、时间序列和案例分析的完整主线。

- `情感得分与治理标签回归说明.md`
  说明如何将连续/离散情感结果与 11 个二级治理标签接入多元线性回归和逻辑回归。

- `交通主题时间序列建模说明.md`
  说明交通主题所使用的两段北京交通拥堵指数数据、SARIMA 建模和 Meta-LSTM-Attention 建模口径。

- `北京动物园综合案例分析说明.md`
  说明为什么将北京动物园作为综合案例，以及如何把 POI、动态回归和多元时间序列探索模型统一起来解释。
